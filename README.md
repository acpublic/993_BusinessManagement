# 第1部：マネジャーの役割と心構え キーワード集

## 1. 現代のビジネス環境
* **VUCA**：変動性（Volatility）、不確実性（Uncertainty）、複雑性（Complexity）、曖昧性（Ambiguity）の頭文字をとった、予測困難な状況を指す言葉です [1, 2]。
* **DX（デジタルトランスフォーメーション）**：デジタル技術による業務変革です。コスト削減や効率化に加え、特定の個人に依存しない「脱属人化」による競争力強化を図ります [1, 3]。
* **SDGs（持続可能な開発目標）**：環境・社会課題の解決と経済成長の両立を目指す国際目標です [4]。実態が伴わないのに取り組んでいるように見せることは「SDGsウォッシュ」と呼ばれ、企業イメージを損ないます [2, 4]。
* **ESG投資**：環境（Environment）、社会（Social）、ガバナンス（Governance）の3要素を重視する投資で、企業の長期的な持続可能性を評価する指標です [2, 4]。

## 2. マネジャーの本質と役割
* **マネジャーの定義とミッション**：1人以上の部下を持つチームを管理する者を指します [2]。ミッションは「チームとしての成果」を出すことです [5]。
* **マネジャーの5つの役割**：
    1. 経営方針とチーム目標の共有・浸透 [5]
    2. 戦略の策定・実行と問題解決による目標達成 [5]
    3. チームのエネルギー（モチベーション）の最大化 [5]
    4. 部下の育成と評価 [5]
    5. リスクの迅速かつ適切な評価・対処 [5, 6]
* **顧客思考**：顧客の要望に応える姿勢ですが、単に従うのではなく、自社が提供すべき適切な範囲を明確に定めることが重要です [1]。

## 3. マネジャーに求められる資質・心構え
* **真摯さ（インテグリティ）**：マネジャーに不可欠な根本的資質です。業務、部下、そして自分自身に対して誠実に向き合う姿勢を指します [6]。
* **合理的な判断**：過去の成功体験や個人のプライド、感情に流されず、現実を直視して客観的に判断することです [6]。
* **リスクへの意欲**：単にリスクを回避するだけでなく、成長の機会がある場合には積極的にそのリスクを取り込む姿勢も必要です [6, 7]。

# 第2部 第1章：マネジャー自身のマネジメント キーワード集

## 1. 自己分析と対人関係
* **ジョハリの窓**：対人関係における自己分析モデル。「開放」「盲点」「秘密」「未知」の4つの領域で構成され、他者からの指摘を受け入れたり自己開示を行ったりすることで「開放領域」を広げ、相互理解を深めます [1], [2]。

## 2. タイムマネジメントと業務遂行
* **仕事の選別（重要度・緊急度のマトリクス）**：仕事を「重要度」と「緊急度」で4象限に分類する手法。マネジャーは「緊急ではないが重要な仕事」に注力し、「重要ではないが急ぎの仕事」は可能な限り部下へ委ねることが求められます [3]。
* **権限委譲**：部下の成長を促すために業務を委ねること。部下を信頼して任せる姿勢が重要ですが、最終的な責任はマネジャー自身にあることを自覚しておく必要があります [3], [2]。

## 3. セルフケアとメンタルヘルス
* **ストレスコーピング**：ストレス要因（ストレッサー）への対処行動。問題を直接解決する「問題焦点型」と、リラックスなどで感情を整える「情動焦点型」の2種類があります [4], [5]。
* **認知行動療法**：ストレスを生む原因となる「認知の歪み（決めつけなど）」を修正し、ポジティブな感情や行動に繋げる心理的なアプローチです [5]。

## 4. 自己研鑽と成長
* **経験学習**：日々の経験を単なる出来事で終わらせず、「内省（振り返り）」を通じて教訓を引き出し、次の行動に応用して学習に繋げるプロセスです [5], [6]。
* **メタ認知**：自分の認知活動（考え方や行動）を客観的に把握し、評価・制御すること。自分を一段高い視点から俯瞰し、冷静な判断や活動の修正に役立てます [7], [8], [9]。

# 第2部 第2章 コミュニケーションの重要性：重要キーワードまとめ

## 1. コミュニケーションの基本
* **コミュニケーションの定義**  
  相互に知覚・感情・思考などを伝達し合うことであり、相手を理解し共感することを含みます [1]。自分と相手は異なる認識を持っているという前提に立つことが大切です [1]。
* **バーバル・コミュニケーション**  
  言語によるコミュニケーションで、話し言葉のほかメールやメモも含まれます [1]。
* **ノンバーバル・コミュニケーション**  
  態度、表情、声色、動作などの言語以外の方法によるもので、受け手の感性に大きな影響を与えます [1]。
* **メラビアンの法則**  
  情報の受け手に与える影響力は、視覚情報が55%、聴覚情報が38%、言語情報が7%であるとする報告です [1]。

## 2. ノンバーバル・コミュニケーションの4分類 [2]
* **パーソナル・スペース**  
  他人に近づかれると不快に感じる空間のことです。
* **セルフ・プレゼンテーション**  
  髪型や衣服などの見た目のことで、マネジャーは信頼に値する外見を心がける必要があります。
* **ボディ・ランゲージ**  
  表情、態度、身振り手振りなどのジェスチャーです。
* **パラランゲージ**  
  声の大きさ、高低、調子、スピードなどの周辺言語です。

## 3. 会議の生産性向上
* **アジェンダ**  
  会議の目的や検討課題などを簡潔に記した書面のことです [3, 4]。
* **ファシリテーター**  
  会議の中立的な進行役で、発言しやすい場を作り、合意形成を支援します [4]。
* **議事録**  
  決定事項や責任の所在を明確にし、不参加者と情報を共有するための記録です [4]。

## 4. コミュニケーションに役立つ理論
* **EQ（感情の知能指数）**  
  自分や相手の感情を理解し、適切にコントロールする能力です [5]。識別・利用・理解・調整の4つの能力から構成されます [5, 6]。
* **エゴグラム**  
  自我の状態を「親（厳しい・優しい）」「大人」「子供（無邪気・従順）」の5つに分類する性格診断法です [6]。
* **ソーシャル・スタイル理論**  
  自己主張と感情表出の強弱により、人を「アナリティカル」「ドライビング」「エミアブル」「エクスプレッシブ」の4タイプに分類する理論です [7]。

# 第2部 第3章 部下のマネジメント：重要キーワードと要点

## 1. 部下に対するアプローチと心構え
* **自律性と自発性の尊重**  
  部下の意見を尊重し、自ら考え実行させることで、信頼関係と成長を促します [1]。
* **廉潔（れんけつ）感と誠実さ**  
  チームの成果を部下と分かち合い、目標に対して真摯に向き合う姿勢が、リーダーとしての信頼の源泉となります [1]。
* **公私のけじめ**  
  公務と私事を明確に区別することで、組織の緊張感を保ち、コンプライアンス違反や不正を未然に防ぎます [2]。
* **「怒る」と「叱る」の違い**  
  感情的に相手を責める「怒る」ではなく、相手の存在価値を認めた上で、改善すべき言動を強く戒める「叱る」姿勢が重要です [2]。

## 2. 業務指示と報告の徹底
* **簡潔明瞭な指示**  
  指示内容は短く分かりやすく伝え、必ず部下に内容を復唱・確認させ、可能な限り記録（文書）として残します [2, 3]。
* **業務機能の明確化と具体化**  
  業務の役割（機能）を明確にし、さらに「何を・どうすべきか」という具体的な作業レベルまで落とし込んで指示します [4]。
* **報告の重要性（5W1H）**  
  仕事は報告をもって完結します。客観的な事実（5W1H）に基づいた報告を求め、特にアクシデント報告は最優先で行わせるよう徹底します [4]。
* **Face to Faceの重視**  
  チャット等では周辺情報が欠落しやすいため、可能な限り対面での報告機会を設け、部下の状況を深く把握する工夫が必要です [3]。

## 3. リーダーシップの発揮
* **PM理論**  
  リーダーシップを「目標達成機能（P機能）」と「集団維持機能（M機能）」の2つの能力要素で捉え、自身のタイプを客観視し向上させる手法です [5]。
* **SL理論（状況対応型リーダーシップ）**  
  部下の成熟度やタイプに合わせて、「指示的行動」と「援助的行動」のバランスを柔軟に変えて接する理論です [6]。

# 第2部 第3章 部下のマネジメント：重要キーワードと要点（後半）

## 1. 部下のモチベーションを高める（同期付け）
* **モチベーション（同期付け）**
  人の内側にある「欲しい」という気持ち（動因：ドライブ）と、外側にある「目標」（誘因：インセンティブ）の2つの要因で構成されます [1]。
* **マズローの欲求段階説**
  人間の欲求を、生理的・安全・社会的・自尊（承認）・自己実現の5段階の階層で捉える理論です [1]。下位の欲求が満たされると、より上位の欲求へと向かいます [2, 3]。
* **ハーズバーグの二要因理論**
  満足に関わる「動機付け要因」（達成感、仕事内容など）と、不満足に関わる「衛生要因」（給与、労働条件など）に分けて考えます。衛生要因を整えるだけでは不満を防ぐのみで、やる気を高めるには動機付け要因へのアプローチが必要です [2, 4]。
* **マグレガーのXY理論**
  「人間は本来仕事が嫌いである」と考えるX理論と、「仕事は本来嫌いなものではない」と考えるY理論です。Y理論に基づき期待をかけることで成果が出る「ピグマリオン効果」などが知られています [4, 5]。
* **自己決定理論**
  「自律性」「有能さ」「関係性」の3つの心理的欲求が満たされると、内発的動機付けが促進されます [6]。

## 2. 多様な人材のマネジメント（ダイバーシティ）
* **ダイバーシティ・マネジメント**
  個々の違いや立場を尊重し、個性を活かすことで変化への適応力を高める考え方です [7]。
* **障害者雇用**
  法定雇用率制度があり、民間企業の法定雇用率は現在2.5%です [7, 8]。マネジャーには、全てのメンバーの潜在能力を発揮させる姿勢が求められます [8]。
* **世代間の理解（Z世代など）**
  SNSの活用や多様性・タイパ（効率）を重視するなどの特徴を理解しつつ、チーム共通の目的を見失わないことが重要です [8, 9]。
* **外国人雇用とコンテクスト文化**
  日本のような「言わなくてもわかる」高コンテクスト文化に対し、低コンテクスト文化の外国人には、曖昧さを排除した具体的で明確な指示が必要です [8-10]。

# 第2部 第4章 上司・外部とのコミュニケーション：重要キーワードと要点

## 1. 上司とのコミュニケーション
* **迅速な報告（特にバッドニュース）**
  アクシデント情報は、組織の不祥事に発展するのを防ぐため、手元に留めず直ちに報告する必要があります [1]。
* **面談による報告の重視**
  経営や人事に関わる重要な問題を含むことが多いため、メールだけでなく短時間でも直接会って話すことが推奨されます [1]。
* **相談事項の明確化**
  単なる結果報告に留まらず、上司に相談したいことや支援してほしい内容をあらかじめ用意しておくことが大切です [1]。

## 2. 外部とのコミュニケーションにおける役割
* **人的ネットワークの構築**
  外部からの協力を得るため、日常から密なコミュニケーションを築いておく役割です [1]。
* **情報発信者（代弁者）**
  チームの目標達成のために外部へ根回しを行い、活動成果を周知させる役割を担います [1, 2]。
* **情報の整理・調整**
  外部からの情報をそのままチームに流すのではなく、混乱を避けるために適切に整理・調整して伝達します [2, 3]。

## 3. 交渉に役立つ心理的効果とテクニック
* **ブーメラン効果**
  信頼関係がない相手を無理に説得しようとすると、逆に強い反発を招く現象です [4, 5]。
* **スリーパー効果**
  時間の経過とともに「誰が言ったか（不信感）」を忘れ、「内容」だけが記憶に残る効果です [4, 5]。
* **返報性の原理**
  相手に譲歩や好意を示すと、相手もお返しをしなければならないと感じる心理作用です [4, 6]。
* **交渉テクニック**
  「ローボール・ハイボール（条件の上げ下げ）」や「グッドコップ・バッドコップ（二人組での役割分担）」などがあります [4, 6]。

## 4. 行動経済学の活用
* **プロスペクト理論**
  人は利益を得る喜びよりも、損失を避けることを優先する（損失回避）傾向があるという理論です [7]。
* **アンカーリング効果**
  最初に提示された情報（価格など）が、その後の判断の基準（イカリ）になる効果です [7, 8]。
* **サンクコスト（埋没費用）**
  すでに支払って回収できない費用に執着し、非合理な判断を継続してしまうことです [8]。
* **現状維持バイアス**
  変化によるリスクを避け、慣れ親しんだ現状を維持しようとする心理傾向です [8, 9]。

# 第1部：マネジャーの役割と心構え キーワード集

## 1. 現代のビジネス環境
* **VUCA**：変動性（Volatility）、不確実性（Uncertainty）、複雑性（Complexity）、曖昧性（Ambiguity）の頭文字をとった、予測困難な状況を指す言葉です [1, 2]。
* **DX（デジタルトランスフォーメーション）**：デジタル技術による業務変革です。コスト削減や効率化に加え、特定の個人に依存しない「脱属人化」による競争力強化を図ります [1, 3]。
* **SDGs（持続可能な開発目標）**：環境・社会課題の解決と経済成長の両立を目指す国際目標です [4]。実態が伴わないのに取り組んでいるように見せることは「SDGsウォッシュ」と呼ばれ、企業イメージを損ないます [2, 4]。
* **ESG投資**：環境（Environment）、社会（Social）、ガバナンス（Governance）の3要素を重視する投資で、企業の長期的な持続可能性を評価する指標です [2, 4]。

## 2. マネジャーの本質と役割
* **マネジャーの定義とミッション**：1人以上の部下を持つチームを管理する者を指します [2]。ミッションは「チームとしての成果」を出すことです [5]。
* **マネジャーの5つの役割**：
    1. 経営方針とチーム目標の共有・浸透 [5]
    2. 戦略の策定・実行と問題解決による目標達成 [5]
    3. チームのエネルギー（モチベーション）の最大化 [5]
    4. 部下の育成と評価 [5]
    5. リスクの迅速かつ適切な評価・対処 [5, 6]
* **顧客思考**：顧客の要望に応える姿勢ですが、単に従うのではなく、自社が提供すべき適切な範囲を明確に定めることが重要です [1]。

## 3. マネジャーに求められる資質・心構え
* **真摯さ（インテグリティ）**：マネジャーに不可欠な根本的資質です。業務、部下、そして自分自身に対して誠実に向き合う姿勢を指します [6]。
* **合理的な判断**：過去の成功体験や個人のプライド、感情に流されず、現実を直視して客観的に判断することです [6]。
* **リスクへの意欲**：単にリスクを回避するだけでなく、成長の機会がある場合には積極的にそのリスクを取り込む姿勢も必要です [6, 7]。
